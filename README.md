# Purushothaman Manivannan — Research Portfolio

Personal academic website for Purushothaman Manivannan, a computational materials science researcher working on DFT, GW–BSE, electronic structure, excitons, two-dimensional materials, and transparent conductors.

## Website

[purushothaman-98.github.io](https://purushothaman-98.github.io/)

## Structure

- `index.html` — academic research portfolio
- `trends.html` — separate writing page
- `style.css` — portfolio styling
- `penguin-explainer.png` — illustration used by the writing page

## Deployment

GitHub Pages serves the static site from the `main` branch. No build step or external framework is required.

## Updating the site

Edit the HTML or CSS, preview locally in a browser, and merge changes into `main`. GitHub Pages will publish the updated files automatically.
